# ⚔️ GENERALS OF THE SHADOWS - Site da Aliança

Bem-vindo ao repositório oficial do site da aliança **GENERALS OF THE SHADOWS** para **Supremacy 1942**!

## 🎯 Objetivo
Site completo para gerenciar informações da aliança, membros, rankings, eventos e candidaturas.

## 📊 Informações da Aliança
- **Nome**: GENERALS OF THE SHADOWS
- **Jogo**: Supremacy 1942
- **Membros**: 11 ativos
- **Tema**: Preto + Roxo

## 🏗️ Estrutura do Projeto

```
generals-of-the-shadows/
├── frontend/           # React - Interface do usuário
├── backend/            # Node.js + Express - API
├── database/           # Scripts SQL
├── docs/               # Documentação e tutorials
└── README.md           # Este arquivo
```

## 🚀 Como Começar

### Pré-requisitos
- Node.js (v16+)
- npm ou yarn
- Git

### Instalação Rápida

1. **Clone o repositório**
```bash
git clone https://github.com/CAPuizFelipe017/generals-of-the-shadows.git
cd generals-of-the-shadows
```

2. **Configure o Frontend**
```bash
cd frontend
npm install
npm start
```

3. **Configure o Backend** (em outro terminal)
```bash
cd backend
npm install
npm start
```
